# 백준 11726: 2xn 타일링

https://www.acmicpc.net/problem/11726

## 문제

2×n 크기의 직사각형을 1×2, 2×1 타일로 채우는 방법의 수를 구하는 프로그램을 작성하시오. <br><br>
**예제 입력**  
첫째 줄에 n이 주어진다. (1 ≤ n ≤ 1,000)

```
2
```

**예제 출력**

```
2
```

## 풀이 코드

```python
import sys
n = int(sys.stdin.readline())

dp = [0] * 1001
dp[1] = 1
dp[2] = 2
for i in range(3, n+1):
    dp[i] = (dp[i-2] + dp[i-1]) % 10007
print(dp[n])

```
